# Synthia Vision

Synthia Vision is a standalone, event-aware AI service designed to integrate with Frigate, MQTT, OpenAI, and Home Assistant. This project aims to provide a stateful AI event processor, policy engine, cost tracker, and more.

## Project Structure

- **src/config**: Configuration files.
- **src/mqtt**: MQTT handling logic.
- **src/openai**: OpenAI integration.
- **src/policy_engine**: Policy engine implementation.
- **src/cost_tracking**: Cost tracking and budget management.
- **src/ha_discovery**: Home Assistant discovery and integration.

## Getting Started

1. Set up your environment.
2. Configure MQTT and OpenAI credentials.
3. Run the service.
